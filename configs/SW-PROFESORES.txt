# SW-PROFESORES Configuration

Device role: Access Switch
Model: Cisco 2960
Function: Access switch for the teachers VLAN.

---

## Basic Configuration

```bash
hostname SW-PROFESORES
```

---

## VLAN Configuration

```bash
vlan 20
 name PROFESORES
exit

vlan 99
 name MGMT
exit
```

---

## Trunk Link to MLS-CORE

The `Fa0/1` interface connects this access switch to `MLS-CORE`.

```bash
interface fastEthernet 0/1
 description Trunk to MLS-CORE
 switchport mode trunk
 switchport trunk allowed vlan 20,99
 no shutdown
exit
```

---

## Access Ports for Teacher Devices

Teacher PCs are connected to access ports in VLAN 20.

```bash
interface range fastEthernet 0/2 - 24
 description Teacher access ports
 switchport mode access
 switchport access vlan 20
 spanning-tree portfast
 no shutdown
exit
```

---

## Management Interface

This switch is managed through VLAN 99.

```bash
interface vlan 99
 description Management interface
 ip address 192.168.2.107 255.255.255.248
 no shutdown
exit
```

---

## Default Gateway

Since this is a Layer 2 switch, it uses a default gateway for remote management.

```bash
ip default-gateway 192.168.2.105
```

---

## Expected VLAN Assignment

| Interface        | Mode   | VLAN   |
| ---------------- | ------ | ------ |
| `Fa0/1`          | Trunk  | 20, 99 |
| `Fa0/2 - Fa0/24` | Access | 20     |

## Expected Tests

From a teacher PC connected to this switch:

```bash
ping 192.168.2.1
```

Expected result:

```text
Reply from 192.168.2.1
```

Teacher devices should be able to reach the server VLAN:

```bash
ping 192.168.2.98
```

Expected result:

```text
Reply from 192.168.2.98
```
